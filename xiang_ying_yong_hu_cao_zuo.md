# 响应用户操作

在本节中我们将了解到如何在CMIS支撑的项目中为 ```HTML元素``` 添加事件监听器，用以响应用户的操作。

## 监听事件

要为某个 ```HTML元素``` 添加事件监听器，我们需要在 ```页面主文件``` 中找到相应的 ```HTML标签```， 通过使用 ```DOM Level 0 Event``` 的方式在元素的标签上添加事件监听器。

举例来说，我们要为一个按钮添加点击事件时，可以这样做：

```html
<!-- 点击事件绑定示例 -->
<button onclick={() => {EventHandler}>点击</button>
```
